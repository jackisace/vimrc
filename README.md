# vimrc

``` sh
runtime! debian.vim
filetype plugin indent on
set nobackup
set noundofile
set nowritebackup
set noswapfile
syntax on
set nu
set relativenumber
set autoindent
set shiftwidth=2
set tabstop=2
set splitright
set scrolloff=999
set autoread
set mouse=a
colorscheme habamax
set background=dark
set showmatch

map <F2> :.w !bash<CR>
map <F3> :r!<C-R><C-L><CR>



```
